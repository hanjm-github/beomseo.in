# 범서고등학교 학교 홈페이지 개설 계획

## 1. 시스템 구성 및 기술 스택

### 서버 환경
- **OS**: Ubuntu 24.04 LTS
- **Web Server**: Nginx
- **Database**: MariaDB
- **Backend**: Flask + uWSGI
- **Security/CDN**: Cloudflare (방화벽 및 Turnstile 보안 관리)

### 사용 기술
- **Frontend**: React
- **Backend**: Flask
- **ORM**: SQLAlchemy
- **Analytics**: Google Analytics (개인정보처리방침 수립 필요)

---

## 2. 권한 체계 (Priority)

1. **관리자**: 모든 권한 보유
2. **학생회**: 학생회 게시판 글 작성 권한 보유
3. **교사**: 이름이 **볼드체**로 표시되는 것 외 일반 권한과 동일
4. **학생**: 일반 글쓰기 가능
5. **울산광역시교육청 IP**: 개인정보 보호가 필요한 글 열람 가능
6. **그 외 접속자**: 제한적 접근

---

## 3. 회원가입 정책

- **가입 제한**: 울산광역시교육청 IP 대역에서만 회원가입 가능
- **기본 권한**: 가입 시 기본적으로 '학생' 권한 부여 (교사, 학생회 계정은 관리자가 수동으로 권한 변경)
- **수집 정보**: 닉네임, 비밀번호 (최소한의 정보로 익명성 보장)

---

## 4. 사이트 기본 구조

- **구조**: 일반 사용자용 페이지 / 관리자용 페이지 분리
- **페이지 유형**:
    1. **게시판형**: 글들이 리스트 형태로 나열
    2. **갤러리형**: 썸네일(사진) 위주로 나열
    3. **특수형**: 게시판/갤러리 형태를 벗어나는 커스텀 페이지
- **공통 기능**:
    - 댓글, 좋아요, 싫어요, 공유 기능
    - 정렬 기능 (좋아요 순, 최신 순)
    - 상단 고정 기능 (관리자가 선택한 글)
- **핵심 공통 기능**:
    - 모든 글은 관리자의 승인 전까지 관리자 외 사용자에게 표시되지 않음
    - 관리자는 글을 승인할 수 있고 승인 해제하여 보이지 않게 할 수 있음
    - 기본적으로 글은 삭제되지 않고 외부에 표시되지 않는 방식으로만 처리됨

## 5. 카테고리 및 페이지 세부 구성

### 🏠 메인 페이지
- 공지사항, 오늘의 급식 등 주요 정보 표시
- 각 메뉴로 쉽게 이동할 수 있는 단축 링크 구성
- **시험 기간 카운트다운 위젯**
    - 메인 페이지에 D-Day 카운트다운 표시
    - 학교 행사 캘린더와 자동 동기화

### 📢 공지사항
- **학교 공지사항**: 관리자 권한 이상 작성 가능 추가
- **학생회 공지사항**: 학생회 권한 이상 작성 가능

### 🗣️ 소통하는 범서고
- **범서고 대신 전해드립니다 (익명 게시판)**
    - 사실상의 자유 게시판 역할
- **동아리원 모집 게시판**
    - 학년별 분리 (1/2학년, 3학년)
    - 리스트에서 상세 클릭 없이 정보 확인 가능 (분야, 기간, 방법, 썸네일 등)
- **선택과목 변경 게시판**
    - 1, 2, 3학년 각 학년별 게시판 분리
    - 리스트에서 '주는 과목', '받는 과목' 즉시 확인 가능
- **학생 청원 게시판**
    - 일정 수 이상의 추천을 받으면 학생회장이 답변
- **방송부 노래 라디오 신청곡**
    - 좋아요 버튼을 통한 신청곡 순위 정렬
    - 매일 새로운 신청곡 접수 (전날 신청곡은 열람만 가능, 수정/투표 불가)
- **설문조사 품앗이**
    - 설문조사 링크를 공유하고 서로의 설문조사에 참여
    - 강제성을 띄게 할 수 있는 방안이 있으면 좋을듯
- **분실물 센터 (Lost & Found)**
    - 습득물 사진 업로드 및 카테고리별 검색 기능.
- **실시간 투표 시스템**
    - 학생회 또는 학급에서 간단한 투표 진행 가능
    - 활용 예시: 축제 공연 투표, 반티 디자인 선정, 체육대회 종목 선호도

### 🏫 학교 생활 정보
- **선생님 교무실 찾기**
    - 교무실 지도 및 좌석 확인 (과목별, 학급별 정렬)
    - 특징/자기소개 문구 한 문장씩 포함 (탑재 허락 필요)
    - **보안**: 교육청 IP 이상 권한에서만 열람 가능
- **범서고 점공 계산기**
    - 입력한 성적 데이터를 바탕으로 교내 석차(예상) 안내
- **내신 등급별 인원표**
- **과목별 수행평가 계획 / 시험 범위**
    - 공지 시 즉시 업데이트
    - 학년별 조회 가능
- **오늘의 급식**
    - NEIS API 연동하여 급식 식단 정보 제공
    - 댓글 및 좋아요/싫어요로 급식 평가 기능
    - 긍정적인 평가만 선별하여 학생회에서 급식실로 의견 전달
- **학교 행사 캘린더**
  - 시험 일정, 학교 행사, 방학 등을 한눈에 볼 수 있는 월간 캘린더 뷰
  - NEIS API 연동하여 학사 일정 정보 제공

### 🖼️ 범서고 갤러리
- **공모전**: 진행 중/완료된 공모전 관리
- **행사 사진**: 기타 학교 행사 사진 업로드 (**보안**: 교육청 IP 이상만 조회 가능)

## 6. 기타 사항
- **다크 모드 지원**: 야간 학습 시 눈의 피로를 줄이는 다크 모드 토글 기능.
- **메인 배너 커스터마이징**: 학생회에서 디자인한 배너를 관리자가 쉽게 교체할 수 있는 기능.
- **PWA (Progressive Web App) 지원**: 모바일에서 앱처럼 설치 가능,푸시 알림 지원
- **알림 시스템**: 댓글, 좋아요, 공지사항 등 알림, 이메일/웹 푸시 선택 가능
- **통합 검색 기능**: 게시판, 갤러리, 급식 등 전체 검색

## 7. 개발 철학
- **가장 중요한 것은 개발의 편리함**
- **두번째로 중요한 것은 학생들의 편리함**
- **목표**:지적 능력과 고운 인성이 조화로운 인간으로 성장시킨다
- **교목**:곰솔나무, 끈기와 장수, 장대함을 상징한다.
- **교화**:모란. 오월에 꼬초가 잎이 풍성하게 피어나 부귀와 번영, 행복을 상징해 범서고의 행복과 번영을 기원하는 뜻에서 교화로 지정하였다.
- **교훈**:지혜로운 눈으로 꿈을 이루고 따뜻한 가슴으로 인류에 봉사하자
